# Vendor Consolidation Proposal

## Background
We currently run three overlapping analytics vendors. Contracts renew in Q3.

## Options Considered
- Keep all three (status quo)
- Consolidate to Vendor A
- Consolidate to Vendor B

| Option | Annual Cost | Migration Effort |
| --- | --- | --- |
| Status quo | $240k | none |
| Vendor A | $130k | 3 weeks |
| Vendor B | $110k | 6 weeks |

## Recommendation
Consolidate to Vendor A. It cuts cost 46% with the lowest migration effort and covers all current use cases.

## Success Criteria
- Single source of truth for analytics within one quarter
- No loss of existing dashboards
- Net annual saving above $100k
